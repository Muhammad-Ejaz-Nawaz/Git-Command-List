Git Command List
Created by Muhammad Ejaz Nawaz Attari Madani
  1. For Folder Initialise

    2. git init

  2. For Cloning folder or Project from Github

    3. git clone (folder url like https://username.github/project)

  3. For Config User Name

    4. git config user.name "Username"

  4. For Config Global User Name

    5. git config --global user.name "Username"

  5. For Config Email

    6. git config user.email "email@email.com"

  6. For Config Global User Name

    7. git config --global user.email "email@email.com"

  7. For Automatic Coloring for Git Command Line

    8. git config --global color.ui auto

  8. For Showing Config List

    9. git config --list

  9. For Looking Status

    10. git status

  10. For Clear Screen of Git Bash

    11. clear

  11. For Git Help

    12. git help

  12. For Tacking and Adding Files on Stage Area

    13. git add index.html

    git add --a (for all files)

    git add -A (for all files)

  13. For Untrack files from Stage Aria after Adding Modified Files

    14. git restore --staged filename

  14. For Unstaging File

    15. git reset index.html

    git reset . (for unstaging all files)

  15. For Commit Staged Files

    16. git commit -m "message"

  16. For Direct Commit unstaged Files after Modified

    17. git commit -a -m "message"

  17. For Restore last commited Files after mistake modifing

    18. git restore filnename

    git checkout filename

  18. For restore all files and directories commited after mistake modifing

    19. git checkout -f

  19. For Add all changes made to tracked files & Commit

    20. git commit -am "message"

  20. For All Commit History

    21. git log

  21. For Push Commited files on Github Repositry

    22. git push -u origin main

  22. For Show Branch List

    23. git branch

  23. For Create New Branch

    24. git branch branchname

  24. Fot Switch to branch

    25. git checkout branchname

  25. For Create new branch and switch to it

    26. git checkout -b newbranchname

  26. For Marge Brances to Master Branch

    27. git merge branchname (Yous must Commit after one branch mrger before second branch)

  27. Delete a merged branch

    28. git branch -d branchname

  28. For Delete Branch

    29. git branch -D branchname

  29. For Push Branch to Github

    30. git push origin branchname (Always push final data on master branch)

  30. For Pull data from branch on github

    31. git pull origin branchname

  31. For Show Changes of two Commited files Crrunt working Directory and staged

    32. git diff

  32. For Show Changes of First Commit and last Staged Data

    33. git diff --staged

  33. Show Changes between two commit files

    34. git diff commit1_ID commit2_ID

  34. For File rename and add in stage

    35. git mv oldfilename newfilename

  35. For Delete File and commit to stage

    36. git rm filename

  36. For Remove Chashe of any file from git

    37. git rm --cached filename

  37. For Push Computer Commited Files to github Repositry

    38. git remote add origin (github folder url)
